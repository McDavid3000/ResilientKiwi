GAME CONTROLS:

In game:

A – Move left 
D – Move right 
SPACE – Jump 
Left arrow key – Shoot left 
Right arrow key – Shoot right 
Up arrow key – Shoot up 
Escape – Pause/Un-pause 

Xbox 360 controller alternatives: 

D-Pad (left and right) – Movement  
A – jump 
Y – shoot up 
X – shoot left 
B – shoot right 
Start – Pause/Un-pause 

Keyboard and mouse controls for game over menu and main menu:

Left click – Select option 
Escape – Quit game 

Xbox controller controls for game over menu and main menu:

A - restart
B - quit

---------------------------------------------------------------------------------------------------------------------------------------------------------------------- 

Running game with debug features: 

F1: Debug mode

F2: HiVis invincibility

Debug mode provides:

	FPS info
	Grid mesh view
	X and Y coordinates of player
	X and Y velocities of player
	Current PlayerState and player Direction
